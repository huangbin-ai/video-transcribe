# Workflow: 视频转文字

将视频或音频转录成完整文字稿。

## 使用方式

用户提供一个视频 URL 或本地文件路径，运行转录工具：

### 在线视频

```bash
./run.sh -i "视频URL"
```

### 本地文件

```bash
./run.sh -i "文件路径"
```

## 处理策略

1. **字幕优先** — 先用 yt-dlp 尝试抓取现成字幕（最快，零 API 消耗）
2. **音频转录** — 没有字幕时，下载音频，用 Gemini 或 mlx-whisper 转录
3. **引擎选择** — Gemini（默认，有标点，需网络）/ mlx（离线，无标点，Apple Silicon）

## 输出

转录结果保存到 `~/Downloads/transcript_时间戳.txt`。

完成后展示前 500 字预览，询问用户是否需要进一步提炼（摘要、笔记、改写）。

## 引擎对比

| 引擎 | 标点 | 网络 | 费用 | 速度 |
|------|------|------|------|------|
| Gemini 2.5 Flash | ✅ 有 | 需要 | 免费额度内零成本 | 快 |
| mlx-whisper | ❌ 无 | 不需要 | 完全免费 | 首次慢（下载模型） |
